# CrazyPHP

My crazy framework for creating ultra-fast webapps.

## Project


New project :

```sh
php vendor/kzarshenas/crazyphp/bin/CrazyCommand.php new project
```

Update project :
```sh
php vendor/kzarshenas/crazyphp/bin/CrazyCommand.php update project
```

Delete project :
```sh
php vendor/kzarshenas/crazyphp/bin/CrazyCommand.php update project
```
